# User-Management-CRUD-application

A frontend CRUD-based User Management System built with HTML, CSS, and Vanilla JavaScript. It allows users to add, edit, delete, and search records through a dynamic interface.

## ✨ Features

- Add Users
- Edit Users
- Delete Users
- Search Users
- Dynamic UI Rendering

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
